CASE 01 · 광고대행사 · 채널 기획·운영

벼랑끝의 마케터

대행사의 단발성 구조를 *지속 가능한 자산*으로 재설계한 콘텐츠 채널

Problem

광고대행사의 모든 캠페인 자산이 광고주에 귀속되어 매번 제로 베이스. 광고주 락인 약함, 단발 영업 반복.

Hypothesis

대행사 자체 페르소나 채널이 자산화되면 광고주 락인과 신규 광고 슬롯 카테고리를 동시에 만들 수 있다.

Action

광고주가 바뀌어도 일관된 톤·세계관 유지 가능한 페르소나 채널 기획·런칭. 캠페인을 채널 콘텐츠 흐름에 자연스럽게 통합.

Effect

대행사 자체 청중·콘텐츠 자산 형성. 광고주가 채널의 축적된 청중에 접근하려고 반복 계약하는 구조 전환.

CASE 02 · PWA 메이커 · 빌드 인 퍼블릭

FocusFlow

광고비 0원, *2주 만에 커뮤니티 360명*. 여성 ADHD 사용자를 위한 PWA

Problem

ADHD 사용자가 일반 to-do 앱으로 못 푸는 세 가지 페인 (약 관리·감정 처리·도파민 보상). 광고 예산 없음.

Hypothesis

본인 페인 + 트위터 자산(2,700) + 시장 트렌드 3축 교집합으로 좁고 깊은 타겟 진입. 빌드 인 퍼블릭으로 마케팅과 검증 동시 진행.

Action

매일 개발 과정 트위터 공유. 인게이지먼트로 기능 가설 실시간 검증. AI 협업으로 React + Firebase 풀스택 솔로 빌드.

Effect

광고 0원에 2주 만에 커뮤니티 360명. DAU/MAU 32.5% · WAU 61.8% · D+1 잔존률 47.5% 달성.
